Giới thiệu

Mẫu web này là giao diện front-end cho một hệ thống "tạo video bằng AI". Nó lưu API key vào localStorage và gửi yêu cầu tới API endpoint bạn cấu hình (ví dụ server proxy của bạn).

Để hoạt động với API thật, bạn cần triển khai một backend (Express/Lambda/etc.) để gọi dịch vụ tạo video (OpenAI/Runway/Play.ht/StableVideo) và trả về URL video. Mình cũng thêm mã ví dụ backend trong thư mục server/.
